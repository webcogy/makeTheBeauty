# MAKE THE BEAUTY (개인 토이 프로젝트)

#### 샴푸,컨디셔너 쇼핑몰

## skill list

- Javascript
- JSX, CSS
- CSS module
- Node.js
- React
- GraphQL
- Apollo Client

## install list

```
npx create-react-app
yarn add react-router-dom

// css module class 여러개 중첩가능
yarn add classnames
```

## folder structure

```
public/images/ => common, goods, main

src
./layout => 공통 컴포넌트
./layout/header
./layout/footer
./layout/wrapper

./pages  => 각각 페이지 폴더작업 ( JS + CSS 로 완전한 모듈화 )
./pages/home
./pages/order

./util   => 각종 유틸
./util/ScrollTop => 기능 : 라우터 이동 시 항상 스크롤 최상위에서 시작
```
